9.6.1 存储器模型 

图9.12 IA-32的三种存储器模型

平坦存储器模型(Flat memory Model)：内存被组织成单一的、连续的地址空间，称为“线性地址空间”。所有的代码、数据和堆栈均包含在该地址空间内，该空间的字节地址范围为0～232-1。
分段存储器模型(Segmented memory model)：每个程序均使用一组独立的地址空间，每个地址空间就是一个段，段的最大长度为232B。逻辑地址由段选择器和偏移量组成，处理机将逻辑地址透明地转换为线性地址。
实地址模式存储器模型(Realaddress mode memory model)：这是为保持与早期的8086处理机兼容的存储器模式。线性地址空间被分为段，段的最大长度为64KB。线性地址空间的最大长度为220B。

　

9.6.2 虚地址模式 
     IA-32体系结构微处理机的虚拟存储器可以通过两种方式实现：分段和分页。存储管理部件包括分段部件SU和分页部件PU两部分。分段部件将程序中使用的虚地址转换成线性地址。而分页部件则将线性地址转换为物理地址。
